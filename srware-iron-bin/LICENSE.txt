Privacy
---------


Synchronization ("Sign in"-Feature)

If users turn on the optinal synchronization, the selected personal data (e.g. History, Bookmarks) is transmitted to the Google Servers
beside standard log information (e.g. your system’s IP address, local time).


Translation

If users use the "Translate now" function, the content of the website is sent to Google-Translate servers.
No personal informations are transmitted, except standard log information (e.g. your system’s IP address, local time).


Malware Check

The Browser will contact Internet servers periodically to download the most recent "Safe Browsing" list from Google. It comtains known maleware sites that can harm your computer.
There are not transmitted identifying information,  but the server receives standard log information,
including an IP address and one or more cookies.  If the system detects a match with your visited website, your browser will send Google a hashed,
partial copy of the site’s URL so that Google can send more information to your browser.


Update checks

The browser checks plug-ins and extensions sporadically  for updates. 
No personal informations are transmitted, except standard log information (e.g. your system’s IP address, local time).

General

As a browser, this product is designed to communicate with the internet. E.g. when you surf to a web page, enter search requests etc., your IP-Adress and/or data you enter will be transmitted to external servers.  Foreign servers data protection policy is not in our hand.  In cases, requests are done via our own server (preinstalled search providers except Google and DuckDuckGo run over our srware-search.com servers and are then purely redirected to the search provider) we track/record no data .

Web search

The securesearch and new tab search services provided herein are provided by Wisesearches Search and powered by Wisesearches search partners. For information on the web search services data collection, please visit http://info.wisesearches.com/privacy-policy/, and the Bing privacy policy ( https://privacy.microsoft.com/privacystatement).
For other search engine, please see: https://www.startpage.com/en/privacy-policy, DuckDuckGo: https://duckduckgo.com/privacy

Privatsphäre
---------


Synchronisation ("Anmelden"-Funktion)

Wenn Nutzer die optionale Synchronisierung einschalten, werden die ausgewählten personenbezogenen Daten (z.B. Verlauf, Lesezeichen) an die Google-Server übermittelt
neben Standardprotokollinformationen (z. B. die IP-Adresse Ihres Systems, Ortszeit).


Übersetzung

Wenn Nutzer die Funktion „Jetzt übersetzen“ nutzen, werden die Inhalte der Website an die Server von Google-Translate gesendet.
Es werden keine personenbezogenen Daten übermittelt, außer Standardprotokollinformationen (z. B. die IP-Adresse Ihres Systems, Ortszeit).


Malware-Check

Der Browser kontaktiert regelmäßig Internetserver, um die neueste "Safe Browsing"-Liste von Google herunterzuladen. Es enthält bekannte Malware-Sites, die Ihrem Computer schaden können.
Es werden keine identifizierenden Informationen übertragen, aber der Server erhält Standardprotokollinformationen,
einschließlich einer IP-Adresse und einem oder mehreren Cookies. Erkennt das System eine Übereinstimmung mit Ihrer besuchten Website, sendet Ihr Browser an Google eine gehashte,
teilweise Kopie der URL der Website, damit Google weitere Informationen an Ihren Browser senden kann.


Update Check

Der Browser prüft sporadisch Plug-Ins und Erweiterungen auf Updates.
Es werden keine personenbezogenen Daten übermittelt, außer Standardprotokollinformationen (z. B. die IP-Adresse Ihres Systems, Ortszeit).

Allgemeines

Als Browser ist dieses Produkt für die Kommunikation mit dem Internet ausgelegt. Z.B. beim Surfen auf einer Webseite, Eingabe von Suchanfragen etc. werden Ihre IP-Adresse und/oder die von Ihnen eingegebenen Daten an externe Server übermittelt. Die Datenschutzrichtlinie für ausländische Server liegt nicht in unserer Hand. In Fällen, in denen Anfragen über unseren eigenen Server erfolgen (vorinstallierte Suchanbieter außer Google und DuckDuckGo laufen über unsere srware-search.com-Server und werden dann rein an den Suchanbieter weitergeleitet) verfolgen/speichern wir keine Daten.

Informationen zur Datenerhebung durch Websuchdienste finden Sie unter http://info.wisesearches.com/privacy-policy/ und in der Datenschutzerklärung von Bing (https://privacy.microsoft.com/privacystatement).
